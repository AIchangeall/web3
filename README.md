# 链聘 ChainHire · Web3 行业全职能招聘平台

一个 web3 / crypto 行业全职能招聘网站，覆盖工程研发、运维 SRE、产品、设计、数据 AI、安全、
市场增长、BD、合规 HR 等各类岗位，跨头部交易所、公链/L2、钱包/DeFi、节点 RPC 基础设施、
合规托管等公司类型，**每 5 小时自动同步职位**（每日多次）。包含职位、数据洞察、企业库、招聘线索四个页面，均按职能 / 公司类型 / 级别 / 地区维度组织。页面顶部显示精确到分钟的最近更新时间。

## 文件结构

| 文件 | 作用 |
|------|------|
| `index.html` | 主页（职位列表）。品牌导航 + Hero 搜索 + 侧栏筛选（职能/公司类型/级别/地区）+ 卡片式职位 + 排序 + 加载更多 + KPI + 更新日志。 |
| `stats.html` | 数据洞察页。KPI、新发布趋势（SVG）、职能分布、公司类型占比（甜甜圈）、Top 招聘公司、级别/地区分布、各职能薪资中位。纯 CSS/SVG 图表，无外部依赖。 |
| `companies.html` | 企业库页。各招聘公司简介 + 基于在招职位实时计算的分析（在招数、薪资中位、远程占比、主要职能、最新发布），点击直达该公司职位。公司简介存于 `data.js` 的 `companies` 字段。 |
| `headhunter.html` | 招聘线索页。线索表：招聘公司 / 岗位信息 / 关键词（技能+职责）/ 来源渠道（按链接自动识别）/ 联系人 / 联系方式 / 最新发布，可搜索、筛选、排序。 |
| `data.js` | 岗位数据源（JSON 风格）。由每日更新程序自动维护，前端 `<script>` 加载。每条含 `func`（职能）与 `category`（公司类型）两个分类维度。 |
| `apply_update.mjs` | 确定性更新引擎：把当日抓取的 `new_jobs.json` 合并进 `data.js`（去重 / 标记新增 / 写日志 / 自动派生岗位关键字 / 补公司占位）。 |
| `keywords.mjs` | 岗位关键字词典与提取器，从职位名+要点自动派生技能关键字，供职位卡片、招聘线索展示与数据洞察的热门关键字分析。 |
| `UPDATE_RUNBOOK.md` | 每日更新操作手册 + 现成的自动化 Agent 提示词。 |

## 打开网站

直接用浏览器打开 `index.html` 即可（双击或拖入浏览器）。若浏览器拦截本地脚本，启动一个本地静态服务：

```bash
cd web3            # 进入项目目录
python3 -m http.server 4173
# 浏览器访问 http://localhost:4173/
```

## 每日更新怎么跑

见 [`UPDATE_RUNBOOK.md`](UPDATE_RUNBOOK.md)。核心三步：联网检索新岗位 → 写 `new_jobs.json` → `node apply_update.mjs new_jobs.json`。

更新引擎特性：按"公司+职位"去重、自动给新岗位标 `firstSeen`（前端显示 NEW 并置顶）、当天重复运行幂等、不会误删旧岗位。

## 数据说明

数据来自公开渠道（公司官网 / Lever / Ashby / Greenhouse / LinkedIn / V2EX / Web3.career / CryptoJobsList / 登链社区）。
薪资为公开 JD 标注或聚合估算，仅供参考，具体以面试谈薪为准。
