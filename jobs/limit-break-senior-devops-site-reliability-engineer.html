What You'll do: Design, implement, and maintain backend services for a live mobile game using Go, gRPC, REST, protobuf, MySQL, Redis, and service-to-service APIs. Build systems that are safe under liveops pressure: remote config, event launches, timed campaigns, content updates, rollbacks, partial failures, and client/server version mismatches.
